SOC Monitoring Dashboard
The BioEcho Shield SOC Dashboard provides a real-time overview of probe events, security alerts, and threat levels detected by the intrusion detection system. It displays key metrics such as total events, total alerts, severity distribution, and system status to help security analysts quickly assess potential threats.

Probe Events and Security Alerts
This section of the dashboard displays detailed probe event logs and generated alerts. Each event contains information such as the event ID, type of activity detected, and timestamp. The alert panel classifies the events based on severity levels and highlights potential intrusion attempts for further investigation
